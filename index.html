jQuery Only Number

Plugin para permitir digitar somente números em um campo

Requisitos

jQuery JavaScript Library v2.1.4

Instalação

Inserir o plugin abaixo do jQuery

Exemplo

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Testando JQuery Only Number 1.0</title>
        <script src="../bower_components/jquery/dist/jquery.min.js"></script>
        <script src="../jquery.onlynumber.js"></script>
    </head>
    <body>


    </body>
</html>

Usando

Insira o comando abaixo para disparar o evento em um input

$('input').onlynumber();

Caso necessário destuir o evento 

$('input').onlynumber("destroy");

Cuidado: certificar sempre se o DOM foi totalmente carregado

$(document).ready(function(){
    $('input').onlynumber();
});

Exemplo

Inserindo o evento ao input valores:

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Testando JQuery Only Number 1.0</title>
        <script src="bower_components/jquery/dist/jquery.min.js"></script>
        <script src="jquery.onlynumber.js"></script>
        <script>
            $(document).ready(function(){
                $("#valores").onlynumber(); 
            });
        </script>
    </head>
    <body>
        <input type="text" id="valores" value="" />
    </body>
</html>

Exemplo 2

Ativando e desativando o plugin usando o parametro destroy

<!DOCTYPE html>
<html lang="en">
    <head>
    <title>Testando JQuery Only Number 1.0</title>
    <script src="bower_components/jquery/dist/jquery.min.js"></script>
        <script src="jquery.onlynumber.js"></script>
        <script>
            $(document).ready(function(){

                $("#ativadesativa").click(function(){

                    var thisObj = $(this);
                    var ativo   = $(this).data("ativo");

                    //Se ativo desativa
                    if (ativo) 
                    {
                        thisObj.data("ativo",0).html("Ativar");
                        $("#valores").onlynumber("destroy"); //Destruindo

                    }
                    else //Se desativado ativa
                    {
                        thisObj.data("ativo",1).html("Desativar");
                        $("#valores").onlynumber(); //Ativando
                    }

                    $("#valores").focus();
                });
            });
        </script>
    </head>
    <body>
            <button type="button" id="ativadesativa" data-ativo="0" >Ativar</button>
        <input type="text" id="valores" value="" />
    </body>
</html>
